🤖 AI Resume Analyzer

🚀 Overview

AI Resume Analyzer is a web application that analyzes resumes and evaluates how well they match a selected job role. It uses basic Natural Language Processing (NLP) techniques to extract and compare keywords from resumes.

---
🧠 Features
- 📄 Upload resume (PDF format)
- 🎯 Select job role (Web Developer, Data Analyst, Cyber Security)
- 📊 Get match score percentage
- ⚡ Fast and simple analysis
- 🖥️ Clean and minimal user interface
---
⚙️ Tech Stack

- Backend: Python, Flask
- Frontend: HTML, CSS
- Libraries: PyPDF2 (PDF parsing), NLP (keyword matching)

2️⃣ Install dependencies
pip install -r requirements.txt
3️⃣ Run the application

4️⃣ Open in browser
http://127.0.0.1:5000

---
📊 How It Works

1. User uploads a resume
2. System extracts text from PDF
3. Selected job role keywords are matched
4. Match score (%) is calculated
5. Result is displayed instantly
