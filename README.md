# AI Resume Analyzer

An AI-powered Resume Analyzer and Builder designed to help job seekers create ATS-friendly resumes, identify skill gaps, and improve their chances of landing interviews.

## 🚀 Overview

AI Resume Analyzer is a modern web application that enables users to:

* Analyze resumes using AI-driven insights
* Calculate ATS (Applicant Tracking System) compatibility scores
* Identify missing skills and improvement areas
* Generate professional resumes with live preview
* Export resumes in multiple formats
* Match resumes with suitable job opportunities

The application provides an intuitive user experience with responsive design, modern UI components, and real-time resume generation.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/prakashalagundagi/Ai-Resume-Analyzer.git
cd Ai-Resume-Analyzer
```

### Run Using Python



### Open Application


## 📖 How to Use

### Resume Analysis

1. Open the Analyzer section.
2. Upload your Resume (PDF/DOC/DOCX).
3. Select target job role.
4. Choose experience level.
5. Click Analyze Resume.
6. View ATS score and recommendations.

### Resume Builder

1. Select a template.
2. Enter personal details.
3. Add education and skills.
4. Add projects and experience.
5. Preview updates instantly.
6. Export the final resume.
